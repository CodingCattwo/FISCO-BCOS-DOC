# 环境要求

```eval_rst
.. important::

    - java版本
     要求 `jdk1.8+ <http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html>`_，推荐使用jdk8u141
    - FISCO BCOS区块链环境搭建
     参考 `FISCO-BCOS搭链教程 <https://fisco-bcos-documentation.readthedocs.io/zh_CN/feature-2.0.0/docs/manual/buildchain.html>`_
    - 网络连通性
     检查Java SDK连接的FISCO BCOS节点channel_listen_port是否能telnet通，若telnet不通，需要检查网络连通性和安全策略。

```

# 社区

![](../images/others/FISCO_logo.png)

[**金链盟**](https://www.fisco.com.cn/)，全称金融区块链合作联盟（深圳），成立于2015年，现有110多家机构成员。金链盟开源工作组，获得金链盟成员机构的广泛认可，并由专注于区块链底层技术研发的成员机构及开发者牵头开展工作。其中首批成员包括以下单位（排名不分先后）：博彦科技、华为、深证通、神州数码、四方精创、腾讯、微众银行、越秀金科。
